## 📂 Описание папок

### `bot/` - Основной код
- **handlers/** - Обработчики команд и сообщений Telegram
- **services/** - Бизнес-логика приложения
- **database/** - Модели данных и подключение к БД
- **keyboards/** - Клавиатуры для Telegram бота
- **middlewares/** - Middleware для обработки запросов
- **states/** - FSM состояния для диалогов
- **utils/** - Вспомогательные функции

### `docs/` - Документация
- **deployment/** - Инструкции по деплою (Docker, VPS и т.д.)
- **testing/** - Руководство по тестированию
- **setup/** - Инструкции по настройке и запуску
- **development/** - Документация для разработчиков

### `specs/` - Спецификации
- Техническое задание проекта

### `scripts/` - Скрипты
- Скрипты для запуска тестов и миграций

### `tests/` - Тесты
- **unit/** - Юнит-тесты отдельных компонентов
- **integration/** - Интеграционные тесты
- **e2e/** - End-to-end тесты полных сценариев
- **fixtures/** - Фикстуры для тестов

### `temp/` - Временные файлы
- Временные файлы, не коммитятся в Git

### `memory-bank/` - Внутренняя документация
- Документация процесса разработки
- Creative фаза документы
- Технические решения
