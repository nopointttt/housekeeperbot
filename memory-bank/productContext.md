# Product Context

## Product Vision
Telegram бот для автоматизации хозяйственной деятельности организации, обеспечивающий прозрачность процессов, ускорение обработки заявок и контроль качества работы завхоза.

## Target Users
1. **Сотрудники** - создают заявки на материалы/работы, отслеживают статус, могут жаловаться
2. **Завхоз** - обрабатывает заявки, управляет складом, делает рассылки
3. **Руководитель** - просматривает заявки, получает жалобы и отчеты, контролирует работу

## Key Features
- Система ролей на основе Telegram ID
- Пошаговый мастер создания заявок с фото
- Управление статусами заявок (Новая → В работе → Выполнено/Отклонено)
- Простая система склада с минимальными остатками
- Автоматические уведомления (напоминания о минимуме, отчеты, срочные заявки)
- Система жалоб с мгновенной отправкой руководителю
- Отчетность и аналитика для руководителя
- Рассылки от завхоза всем сотрудникам

## Business Context
Бот решает проблему неорганизованности в хозяйственной деятельности: заявки теряются, нет контроля сроков, склад не контролируется. Автоматизация процессов повышает эффективность работы завхоза и прозрачность для руководства.
