# Project Brief

## Project Overview
Разработка Telegram бота "Завхоз" для автоматизации управления заявками и складом в организации. Бот обеспечивает взаимодействие между сотрудниками, завхозом и руководителем через единую платформу с системой ролей, отслеживанием заявок, управлением складом и автоматическими уведомлениями.

## Objectives
1. Создать систему управления заявками с пошаговым мастером создания
2. Реализовать систему ролей (Сотрудник, Завхоз, Руководитель) с разграничением прав
3. Внедрить функционал управления складом с автоматическими напоминаниями
4. Организовать систему жалоб и отчетности
5. Автоматизировать уведомления и напоминания

## Scope
### Включает:
- Система ролей и прав доступа (3 роли)
- Создание и управление заявками (9 категорий)
- Управление складом (добавление, списание, минимумы)
- Система жалоб сотрудников
- Автоматические уведомления и напоминания
- Отчетность для руководителя
- Рассылки от завхоза

### Не включает (на текущем этапе):
- Веб-панель администратора
- Интеграции с внешними системами
- Мобильное приложение

## Success Criteria
1. Все роли функционируют с правильным разграничением прав
2. Заявки создаются и обрабатываются без ошибок
3. Склад работает корректно с автоматическими напоминаниями
4. Все автоматические уведомления работают в заданное время
5. Система жалоб и отчетности функционирует
6. Бот стабильно работает 24/7
