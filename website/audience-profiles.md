# Портреты аудитории и Jobs To Be Done (JTBD)

## 1. ОТЕЛИ

### Портрет аудитории (ICP)
- **Тип**: Полносервисные отели, бизнес-отели, городские отели
- **Размер**: 50-500 номеров
- **Локация**: Москва и регионы России
- **Особенности**:
  - Высокие стандарты сервиса
  - Важна репутация и отзывы
  - Постоянный поток гостей
  - Строгие SLA по времени реакции
  - Команда: администраторы, техники, горничные, управляющий
- **Боли**:
  - Жалобы гостей на медленную реакцию
  - Потерянные заявки между сменами
  - Нет контроля выполнения работ
  - Низкие отзывы из-за медленного решения проблем
  - Непрозрачность процессов для руководства

### Jobs To Be Done (JTBD)
1. **Реагировать на жалобы гостей моментально** - чтобы гости были довольны и оставляли хорошие отзывы
2. **Контролировать выполнение всех заявок** - чтобы ничего не терялось и все работало
3. **Повышать качество сервиса** - чтобы увеличить рейтинг на Booking/Оstrovok и повторные бронирования
4. **Снижать количество жалоб** - чтобы сохранять репутацию и не терять клиентов
5. **Видеть полную картину работы** - чтобы принимать обоснованные управленческие решения

---

## 2. ХОСТЕЛЫ

### Портрет аудитории (ICP)
- **Тип**: Бюджетные хостелы, молодежные хостелы, хостелы для путешественников
- **Размер**: 10-100 мест (кроватей)
- **Локация**: Москва, крупные туристические города
- **Особенности**:
  - Очень высокая текучесть гостей
  - Низкая маржинальность
  - Минимальный персонал (1-2 администратора, 1 техник)
  - Частые мелкие поломки из-за высокой нагрузки
  - Важна скорость и простота
- **Боли**:
  - Огромное количество мелких заявок
  - Нет времени разбираться со сложными системами
  - Персонал перегружен
  - Гости уезжают быстро, нужно решать проблемы СРАЗУ
  - Нет бюджета на дорогие решения

### Jobs To Be Done (JTBD)
1. **Быстро обрабатывать множество заявок** - чтобы успеть до отъезда гостей
2. **Простое и понятное решение** - чтобы персонал быстро обучился
3. **Не терять заявки** - чтобы все поломки были устранены
4. **Работать с минимальным персоналом** - чтобы не нанимать дополнительных сотрудников
5. **Быстрая реакция на критические проблемы** - чтобы не потерять репутацию и отзывы

---

## 3. УПРАВЛЯЮЩИЕ КОМПАНИИ (УК)

### Портрет аудитории (ICP)
- **Тип**: УК жилых комплексов, коммерческой недвижимости
- **Размер**: От 3-5 объектов до крупных сетей (50+ объектов)
- **Локация**: Москва, регионы
- **Особенности**:
  - Управление множеством объектов одновременно
  - Централизованное управление техниками
  - Нужна отчетность для собственников
  - Разные объекты, разные требования
  - Важна масштабируемость и контроль
- **Боли**:
  - Нет единой системы для всех объектов
  - Сложно контролировать работу техников на разных объектах
  - Нет прозрачности для собственников
  - Потеря заявок между объектами
  - Сложно собирать статистику и отчитываться

### Jobs To Be Done (JTBD)
1. **Управлять заявками по всем объектам в одном месте** - чтобы контролировать работу централизованно
2. **Видеть полную картину по всем объектам** - чтобы принимать стратегические решения
3. **Прозрачность для собственников** - чтобы показывать качество обслуживания
4. **Масштабируемость** - чтобы легко добавлять новые объекты
5. **Отчетность и аналитика** - чтобы оптимизировать процессы и планировать бюджеты

---

## 4. САНАТОРИИ

### Портрет аудитории (ICP)
- **Тип**: Санатории, пансионаты, дома отдыха
- **Размер**: 50-300 номеров
- **Локация**: Курортные зоны, регионы
- **Особенности**:
  - Длительное пребывание гостей (7-21 день)
  - Медицинские и оздоровительные процедуры
  - Высокие требования к комфорту и тишине
  - Сезонность (пики нагрузки)
  - Команда: администраторы, техники, медперсонал, управляющий
- **Боли**:
  - Гости остаются надолго - проблемы должны решаться быстро
  - Важна тишина и комфорт - поломки критичны
  - Медицинское оборудование требует особого внимания
  - Сложно отслеживать профилактические работы
  - В пиковый сезон перегрузка персонала

### Jobs To Be Done (JTBD)
1. **Обеспечивать комфорт гостей на протяжении всего пребывания** - чтобы они были довольны и возвращались
2. **Быстро реагировать на проблемы** - чтобы не нарушать лечебный процесс
3. **Контролировать профилактические работы** - чтобы предотвращать поломки
4. **Справляться с пиковыми нагрузками** - чтобы не терять качество в сезон
5. **Поддерживать репутацию** - чтобы получать рекомендации и повторные визиты

---

## Общие JTBD для всех типов

1. **Моментальная реакция** - видеть новые заявки сразу
2. **Прозрачность процессов** - понимать, что происходит в любой момент
3. **Снижение жалоб** - решать проблемы до того, как они станут жалобами
4. **Повышение качества сервиса** - улучшать гостевой опыт
5. **Контроль и управляемость** - иметь инструменты для эффективного управления
